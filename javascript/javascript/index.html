# 新建数组

//ES5 新建数组
const arr1 = [] //数组字面量
const arr2 = new Array() //new构建

//ES6
let items = Array.of(1, 2) // 解决怪异行为
let args = Array.from(arguments) //将非数组对象转换为真正的数组

//Array.from(arg1, arg2)，我们可以给该方法提供2个参数，第二个参数作为第一个参数的转换：
function translate() {
  return Array.from(arguments, (value) => value + 1)
}
let numbers = translate(1, 2, 3)
console.log(numbers) // 2,3,4

# 数组操作迭代方法

接收三个参数，数组当前项的值、当前项在数组中的索引、数组对象本身

some 回调函数一个为 True 返回 Boolean
every 回调函数所有为 True 返回 Boolean

some 方法是碰到一个返回 true 的值时候就返回了，并没有继续往下运行，而 every 也一样，第一个值就是一个 false,所以后面也没有进行下去的必要了，就直接返回结果了

# Map 返回每个元素的回调函数的 数组

创建新数组，其中结果是该数组中的每个元素都调用一个提供的函数后返回的结果。map 不会改变原数组。

var names = animals.map((x) => x.name)

# Filter 返回通过回调函数的数组

创建新数组，其中包含所有通过回调函数测试的元素。

var dogs = animals.filter((x) => x.species === "dog")

过滤整个数组时，可以使用 filter()。当确定要搜索数组中的唯一元素时，则使用 find()。

从数组中删除虚值 在 JS 中，虚值有 false, 0，''， null, NaN, undefined。咱们可以 .filter() 方法来过滤这些虚值。

# 数组操作索引方法

  • find 返回回调函数为 True 值
  • findIndex 返回回调函数为 True 索引值 没有-1
  • indexOf 返回在数组中可以找到一个给定元素的第一个索引，如果不存在则返回-1
  • lastIndexOf 返回指定元素在数组中的最后一个索引。
let numbers = [25, 30, 35, 40, 45];
console.log(numbers.find(n => n > 33)); // 35
console.log(numbers.findIndex(n => n > 33)); // 2

numbers.fill(1, 2);

# copyWithin() 方法

选择数组的某个下标，从该位置开始复制数组元素，默认从 0 开始复制。也可以指定要复制的元素范围：

let numbers = [1, 2, 3, 4];
// 从索引 2 的位置开始粘贴
// 从数组索引 0 的位置开始复制数据
numbers.copyWithin(2, 0);
console.log(numbers.toString()); // 1,2,1,2

let numbers = [1, 2, 3, 4];
// 从索引 2 的位置开始粘贴
// 从数组索引 0 的位置开始复制数据
// 在遇到索引 1 时停止复制
numbers.copyWithin(2, 0, 1);
console.log(numbers.toString()); // 1,2,1,4
